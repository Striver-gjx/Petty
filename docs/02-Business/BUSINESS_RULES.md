# Business Rules

## 订单状态流转

```
PENDING_MATCH → PENDING_ACCEPT → ACCEPTED → SITTER_EN_ROUTE → IN_SERVICE → SERVICE_COMPLETED → OWNER_CONFIRMED
```

异常状态：`CANCELLED` / `DISPUTED` / `REFUNDED`

### 状态转换规则

| 当前状态 | 触发动作 | 目标状态 | 触发方 |
|----------|----------|----------|--------|
| PENDING_MATCH | 系统匹配到喂养师 | PENDING_ACCEPT | System |
| PENDING_ACCEPT | 喂养师接单 | ACCEPTED | Sitter |
| PENDING_ACCEPT | 喂养师拒单/超时 | PENDING_MATCH（重新匹配） | Sitter/System |
| ACCEPTED | 喂养师出发 | SITTER_EN_ROUTE | Sitter |
| SITTER_EN_ROUTE | 到达打卡 | IN_SERVICE | Sitter |
| IN_SERVICE | 服务完成打卡 | SERVICE_COMPLETED | Sitter |
| SERVICE_COMPLETED | 主人确认 | OWNER_CONFIRMED | Owner |
| SERVICE_COMPLETED | 超过24h未确认 | OWNER_CONFIRMED（自动确认） | System |
| 任意（服务开始前） | 取消 | CANCELLED | Owner/Sitter |

---

## 匹配规则

### 匹配引擎优先级

1. **可用性过滤**：喂养师必须在该时段有空余排班
2. **地理过滤**：服务地址在喂养师服务半径内
3. **物种过滤**：喂养师接受该类型宠物
4. **排名因子**：
   - 距离（权重 30%）
   - 评分（权重 30%）
   - 完成率（权重 20%）
   - 响应速度（权重 20%）

### 指定喂养师

- 主人可指定上次服务的喂养师（跳过匹配，直接进入 PENDING_ACCEPT）
- 指定喂养师拒单后回到智能匹配流程

## 服务标准

### 必须完成项（否则无法完成打卡）

| 服务类型 | 必须提交 |
|----------|----------|
| 所有类型 | 到达打卡照（含GPS校验） + 离开打卡照 |
| 上门喂养 | 至少 1 条喂食记录 + 宠物状态照 |
| 遛狗 | GPS 轨迹记录 + 至少 2 张户外照 |
| 陪玩 | 至少 1 条互动记录 + 视频/照片 |
| 喂药 | 喂药记录照片 + 宠物反应描述 |

### GPS 校验

- 到达打卡：GPS 距离服务地址 ≤ 200 米
- 离开打卡：GPS 距离服务地址 ≤ 500 米
- 在场时长 ≥ 预约时长 × 80%（否则需要补充说明）

## 取消规则

| 取消时间点 | 费用 | 说明 |
|------------|------|------|
| 预约时间前 24 小时 | 免费取消 | 全额退款 |
| 预约时间前 2-24 小时 | 收取 20% | 退款 80% |
| 预约时间前 2 小时内 | 收取 50% | 退款 50% |
| 服务已开始 | 不可取消 | 可发起纠纷 |

### 喂养师取消

- 喂养师取消不扣主人费用，全额退款
- 喂养师取消率纳入信用评估（超过 10% 降权）
- 系统自动为主人重新匹配

---

## 支付规则

### 资金流转

```
主人下单支付 → 平台托管（预授权） → 服务完成 → 主人确认 → 资金释放（扣佣后打款喂养师）
```

### 佣金抽成

| 喂养师等级 | 平台佣金 | 说明 |
|------------|----------|------|
| 新手（< 10 单） | 25% | 含平台保险 |
| 成熟（10-100 单） | 20% | 含平台保险 |
| 金牌（> 100 单） | 15% | 含平台保险 |

### 提现规则

- 最低提现金额：50 元
- 提现到账时间：T+1 工作日
- 每日最多提现 1 次

---

## 评价规则

### 双向评价

- 服务完成后 72 小时内可提交评价
- 超过 72 小时未评价：系统默认 5 分好评
- 评分维度：准时性、服务质量、沟通态度、宠物对待、推荐意愿
- 单次评价最多上传 9 张图片

### 评分影响

- 综合评分 < 4.0：系统警告
- 综合评分 < 3.5：暂停接单，需培训
- 综合评分 < 3.0：强制下线

---

## 信任与安全

### 喂养师准入

| 审核项 | 要求 |
|--------|------|
| 实名认证 | 身份证正反面 + 人脸核验 |
| 背景调查 | 无犯罪记录证明 |
| 技能证书 | 宠物护理培训证书（推荐但非必须） |
| 培训考核 | 平台标准化服务流程考核（必须） |
| 保证金 | 300 元（可退还） |

### 保险保障

- 每单自动投保：宠物意外险（最高赔付 5000 元）
- 第三方责任险：喂养师造成宠物伤害赔付
- 主人财物损失：最高赔付 2000 元

### 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 宠物状态异常 | 喂养师必须上报 + 联系主人 + 平台介入 |
| 无法进门 | 联系主人 → 15 分钟无回应 → 订单暂停，不扣费 |
| 主人投诉 | 48 小时内平台介入裁决 |
| 喂养师失联 | 系统预警 → 紧急联系 → 派备选喂养师 |

## 会员等级

### 宠物主人等级

| 等级 | 条件 | 权益 |
|------|------|------|
| NORMAL | 默认 | 基础服务 |
| VIP | 累计消费 ≥ 500 或订单 ≥ 10 | 95 折 + 优先匹配 |
| SVIP | 累计消费 ≥ 2000 或订单 ≥ 50 | 9 折 + 专属客服 + 免费取消次数 ×3/月 |

### 喂养师等级

| 等级 | 条件 | 权益 |
|------|------|------|
| 新手 | < 10 单 | 系统导流 + 培训支持 |
| 成熟 | 10-100 单 + 评分 ≥ 4.5 | 降佣 + 优先推荐 |
| 金牌 | > 100 单 + 评分 ≥ 4.8 | 最低佣金 + 首页展示 + 认证标识 |
